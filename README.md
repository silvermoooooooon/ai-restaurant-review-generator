# AI 餐厅点评生成器

这是一个基于 AI 的餐厅菜品点评生成器，可以帮助餐厅主理人根据菜品描述自动生成真实、多样的顾客好评。

## 功能特点

- 🍽️ 支持输入详细的菜品描述
- 🤖 使用 AI 技术生成自然、真实的顾客点评
- ✨ 每次可生成多个不同风格的 100 字好评
- 💡 智能分析菜品特点，突出菜品亮点

## 技术栈

- 前端：React + TypeScript + Tailwind CSS
- 后端：Node.js + Express
- AI：OpenAI GPT API
- 部署：Vercel
